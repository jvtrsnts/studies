# Estudo JAVA

### Java

- Todo programa em Java precisa ter uma classe
  - public class Main:
    - <visibilidade\> class <nome\>
- Todo programa em Java precisa ter uma função
  - public static void main(Strings[] args) {}
    - <visibilidade\> < > <nome\> (<tipo argumento\> <nome do argumento\>, ) {}
    - procura instruções a partir da função main()

### Olá mundo

- System.out.println()
  - System.out = saida padrão
  - .println() = print com quebra de linha
  - Java é case sensitive

<br>

- " " = strings
- ' ' = char

<br>

- Comandos em java terminam com ;
