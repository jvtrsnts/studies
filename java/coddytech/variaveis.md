# Variáveis

- **Sintaxe:** <tipo\> <nome\> = <valor\>;
- Tipos primitivos armazenam o valor real

<br>

### Números

- Tipos: _int_ e _double_
- Ao declarar o tipo de uma variavel, ela só pode receber valores desse tipo(estaticamente tipada)

<br>

### String

- _char_ é m caracter único: ' '
- _String_ é um conjunto de _char_: " "
