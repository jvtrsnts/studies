# Basics JAVA

- Linguagem estaticamente tipada
  - Não pode mudar seu tipo
    int explicitVar = 10

- Orientada a objeto
  class Classe {}

- Função dentro de classes são chamadas de métodos
  - Parametros devem ser tipados explicitamente
  - Retorno devem ser tipados explicitamente

- Recap
  - public: instâncias de outras classes podem chamar
  - protegido: instâncias da mesma classe podem chamar
  - privado: apenas a instância do método que pode chamar

- Instância de classe: new
